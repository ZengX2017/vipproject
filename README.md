# 关于此项目

## 1. 本项目是收集了各种VIP解析所做，功能上来说是可以看大部分的全网VIP视频。

## 2. 此站不提供广告，若在使用中出现广告，可能是ip挟持。

## 3. 操作步骤：
	  1，去相应网站打开要看的视频；
	  2，复制地址栏的链接；
	  3，将链接粘贴至输入框，然后点击->解析播放。
	  提示：如果默认解析不能播放，请更换接口(切换线路)
